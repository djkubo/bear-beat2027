# Demos estáticos (evita 503 en Render)

Los demos se sirven desde aquí para no depender de la API `/api/demo/` (streaming bloqueado en Render).

**Estructura:** Misma que el catálogo. Ejemplo:
- `Bachata/Artista - Título.mp4`
- `Reggaeton/Otro - Video.mp4`

Coloca aquí clips cortos de ejemplo por género. Next.js los sirve como estáticos en `/demos/...`.
