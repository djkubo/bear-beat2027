# Logos "Compatible con" (Serato, Rekordbox, VirtualDJ, etc.)

Para que la sección **"Compatible con el estándar de la industria"** y el hero muestren los **logos oficiales**, añade en esta carpeta (`public/logos/`) los archivos con estos nombres:

| Archivo       | Marca      | Dónde conseguir logos oficiales |
|---------------|------------|----------------------------------|
| `serato.png` o `serato.svg`   | Serato     | [serato.com/resources](https://serato.com/resources) – Recursos / Logo Pack (cuenta Serato) |
| `rekordbox.png` o `rekordbox.svg` | rekordbox  | [rekordbox.com](https://rekordbox.com) / AlphaTheta (Pioneer DJ) – Prensa o soporte |
| `virtualdj.png` o `virtualdj.svg` | VirtualDJ  | [virtualdj.com](https://www.virtualdj.com) – Sección prensa / partner |
| `pioneer-dj.png` o `pioneer-dj.svg` | Pioneer DJ | [pioneerdj.com](https://www.pioneerdj.com) – Recursos de marca / partner |
| `denon-dj.png` o `denon-dj.svg`   | DENON DJ   | [denondj.com](https://www.denondj.com) – Prensa / partner |

- **Formatos:** PNG (fondo transparente) o SVG. El sitio prueba primero `.svg` y luego `.png` si falla.
- **Tamaño recomendado:** altura ~80–120 px (el layout la escala). Proporción libre.
- Si no añades un archivo, se mostrará el nombre de la marca en texto.

No incluyas en el repo logos con restricciones de uso; añádelos localmente o desde los kits oficiales de cada marca.
